Y Combinator's request for startups organized in a searchable dashboard.
Y-combinator has published their Request For Startups (RFS) with the page showing previous RFS up to summer 2024. I'm not reading all of that so I put it in json, & made a dashboard now you can search by domain and query across cohorts 2024-2026

Here's a rough list of domains covered over the years
- Aerospace
- B2B
- Internal tools
- Medicine
- Robotics
- Software
- Space


The project's on github,  open-source under the MIT license

Find the app here if interested

http://ycombinator-rfs.netlify.app
