## What You Need to Know

If you are a current user of Google Antigravity, **you might need to update** your application to start using the new model.

Along with the addition of Gemini 3.1 Pro, Antigravity has rolled out several new features, improvements, fixes, and patches since the start of 2026.

Some notable updates from the new release include:

- **Browser Fixes:** Improved stability during web interactions.
- **Stability and UI Improvements:** A cleaner, more responsive user interface.
- **Fix for Windows Auto-updater:** Ensuring updates are seamless without manual intervention.
